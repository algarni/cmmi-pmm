Task: Obtain Commitment on Process Definition
Develop an understanding and commitment with the process improvement organization stakeholders on the content of the process and that it meets the requirements as well as the reference model standards.
Disciplines: Organizational Process Focus (OPF)
Purpose

The purpose of this task is to formally verify that the results of the process definition tasks conform to the requirements as well as the reference model standards.
